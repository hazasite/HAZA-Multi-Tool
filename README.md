**Advanced Utility Dashboard - Elevated Experience**

HAZA MULTI-TOOL is a powerful, all-in-one desktop utility application designed to streamline your daily workflow. Whether you are a developer, an office worker, or a general user, this tool brings essential utilities into one clean, modern dashboard. 

Designed and developed by **HAZA**.

## тЬи Key Features

### ЁЯУБ 1. Backup Wizard
Safely sync and backup your important data with ease.
* Auto-detect common folders (`desktop`, `documents`, `downloads`).
* Simple and fast source-to-destination directory mapping.

### ЁЯУЭ 2. Quick File Creator
Instantly generate files without navigating through standard file explorers.
* Create any file type (e.g., `index.html`, `script.js`) directly in your target folder.

### ЁЯЪА 3. Environment Profiles (Workspaces)
Launch your entire workspace with a single click to boost productivity.
* **Development:** Instantly opens VS Code, Notepad, and your browser.
* **Entertainment:** Launches YouTube, Facebook, and Netflix for quick breaks.
* **Productivity:** Opens Excel, PowerPoint, and Word simultaneously.

### ЁЯМР 4. Network Tools
Built-in network diagnostics for quick troubleshooting.
* Check internet connectivity (Ping).
* Easily view your IP address with a click of a button.

### ЁЯРЩ 5. GitHub Manager
Manage your GitHub repositories without ever touching the command line.
* **Push to GitHub:** Sync and upload files seamlessly.
* **Clone from GitHub:** Download repositories quickly.
* **Create Release:** Draft release versions effortlessly.
* **Repository Manager:** Create or delete repos directly from the app.
